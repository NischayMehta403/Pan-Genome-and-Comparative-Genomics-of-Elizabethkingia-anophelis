## Genomes used in this project

This project analyzes multiple *Elizabethkingia anophelis* strains.

Due to size constraints, raw genome FASTA files are not stored in this repository.
All genomes were downloaded from NCBI RefSeq / GenBank.

| F3201 | GCF_002022085.1 | Complete |
